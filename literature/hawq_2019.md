# HAWQ: Hessian AWare Quantization of Neural Networks with Mixed-Precision

- **Authors:** Zhen Dong, Zhewei Yao, Amir Gholami, Michael W. Mahoney, Kurt Keutzer
- **Year:** 2019 / ICCV 2019
- **Source:** https://arxiv.org/abs/1905.03696
- **DOI:** https://doi.org/10.1109/ICCV.2019.00038

## Summary

HAWQ proposes Hessian-aware mixed-precision quantization. Instead of assigning the same precision to every layer, it uses second-order information to estimate layer sensitivity and select relative precision levels. The motivation is that brute-force mixed-precision search is infeasible because the design space grows exponentially with model depth.

This is not an LLM paper, but it is a foundational adaptive precision-assignment paper: it provides a principled answer to "which layers deserve more bits?"

## Relevance To This Project

The project's stability-signal policy is a lighter-weight cousin of HAWQ. Rather than estimating a full Hessian spectrum, this project can approximate sensitivity through loss deltas, gradient spikes, activation outliers, clipping/saturation rate, and quantization error from short calibration passes.

## Key Takeaways

- Mixed precision needs a sensitivity metric; otherwise precision assignment becomes ad hoc.
- Higher precision should be reserved for layers where perturbation has larger quality impact.
- Second-order information is a strong but potentially expensive signal.

## Evidence Gaps

- HAWQ targets CNN-style quantization and inference more than LLM training.
- Hessian estimation may be too expensive for a small LoRA pilot.
- It does not directly study BF16/FP16/FP8 floating-point stability or dynamic training signals.
